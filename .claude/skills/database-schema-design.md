# Database Schema Design Skill

Skill especializada em design de esquemas PostgreSQL/Supabase para aplicações SaaS multi-tenant.

## Capacidades

- Design de esquemas relacionais eficientes e escaláveis
- Modelagem de dados para multi-tenancy
- Estratégias de relacionamento entre tabelas
- Normalização vs denormalização
- Design de schemas para SaaS (assinaturas, planos, etc.)
- Otimização de estruturas de dados

## Stack

- PostgreSQL 15+
- Supabase (PostgreSQL host)
- Modelagem de dados relacional
- Multi-tenancy patterns

## Quando Usar

- Criar novos esquemas de banco de dados
- Redesenhar schemas existentes
- Planejar estrutura para novos features
- Implementar multi-tenancy
- Otimizar estrutura de dados

## Exemplo de Uso

/skill database-schema-design --feature "User Management System" --entities users,profiles,roles --multi-tenant true
