# API Design Skill

Skill especializada em design de APIs RESTful usando Next.js Route Handlers.

## Capacidades

- Design de endpoints RESTful
- Definição de schemas de request/response
- Implementação de status codes apropriados
- Versionamento de APIs
- Documentação de APIs
- Rate limiting e segurança
- Error handling padronizado

## Stack

- Next.js Route Handlers
- TypeScript
- Zod (validação)
- OpenAPI/Swagger

## Quando Usar

- Criar novos endpoints da API
- Redesenhar APIs existentes
- Documentar endpoints
- Implementar versionamento
- Adicionar validação de schemas

## Exemplo de Uso

/skill api-design --type REST --resources users,products,orders --version v1
