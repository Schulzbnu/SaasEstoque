# Test Planning Skill

Skill especializada em criação de planos de teste abrangentes.

## Capacidades

- Criação de planos de teste estratégicos
- Definição de cobertura de testes
- Identificação de casos de teste críticos
- Estratégias de teste (unit, integration, E2E)
- Planejamento de testes de performance
- Definição de critérios de aceite

## Tipos de Testes

### Unit Tests
- Testes de funções isoladas
- Testes de componentes React
- Testes de utilitários
- Mock de dependencies

### Integration Tests
- Testes de API Routes
- Testes com Supabase
- Testes de Server Actions
- Testes de banco de dados

### E2E Tests
- Fluxos completos de usuários
- Testes de autenticação
- Testes de formulários
- Testes de checkout

### Performance Tests
- Load testing
- Stress testing
- Lighthouse scores
- Web Vitals

## Quando Usar

- Criar plano de teste para novos features
- Planejar cobertura de testes
- Definir estratégia de testes
- Identificar casos críticos

## Exemplo de Uso

/skill test-planning --feature "User Authentication" --testType unit,integration,e2e --coverage 80
